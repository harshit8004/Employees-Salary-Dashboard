### Employees-Salary-Dashboard (Power BI)
-This project is a Power BI dashboard that analyzes employee salaries across different departments, job roles, locations, and demographics.

-It helps show important insights such as total salary, average salary, top job roles, department-wise salary breakdown, and geographic salary trends.

 ## Project Features
-Total salary, average salary, and employee count

-Salary analysis by department and job title

-Employee distribution by gender and age

-Salary comparison across different locations

-Interactive filters for easy exploration
